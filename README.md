# Cafe Speak Site

## Overview
This repository contains a React front-end application with setup instructions and essential scripts for development.
The project was bootstrapped with Stellar Minore's react template (https://github.com/Stellar-Minore/react-template).

## Prerequisite
- Node.js (version >= v24.1.0)

## Setup Instructions
1. **Clone the repository**
   ```sh
   git clone https://github.com/Stellar-Minore/cafe-speak-site.git
   cd cafe-speak-site
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Run the development server**
   ```sh
   npm start
   ```
   The application will be available at `http://localhost:3000/` by default.

## Scripts
- **Start development server**: `npm start`
- **Build for production**: `npm run build`
- **Run tests**: `npm test`
- **Linting**: `npm run lint`
- **Auto fix lint errors**: `npm run fix-lint`
